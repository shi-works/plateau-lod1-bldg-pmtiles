# plateau-bldg-lod1-pmtiles

## データについて
- 本データは、国土交通省がオープンデータとして公開している、[3D都市モデルPLATEAUの建築物データ（LOD1）](https://www.geospatial.jp/ckan/dataset/plateau)を[tippecanoe](https://github.com/felt/tippecanoe)を[PMTiles形式](https://github.com/protomaps/PMTiles)に変換したデータになります。
- オープンソースソフトウェアで構築

## デモサイト
- https://web-map-maplibre.s3.ap-northeast-1.amazonaws.com/PLATEAU-VIEW/index.html

## ベクトルタイル設計情報
- 建築物データ（LOD1）そのものを可能な限り生かしたデータです。
- tippecanoeによるデータの間引きは行っていません。

### ズームレベル範囲
- 0-14

### 属性
- 建築物データ（LOD1）の属性はそのまま生かしています。

## PMTiles Viewer
- PMTilesはPMTiles Viewerで閲覧することができます。
- https://protomaps.github.io/PMTiles/
